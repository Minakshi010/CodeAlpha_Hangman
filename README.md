# 🎮 Hangman Game (GUI Version)

A simple GUI-based Hangman game built using Python and Tkinter.

## 🚀 Features

- Random word selection
- 6 wrong attempts limit
- Live word display with underscores
- Win/Lose message
- Restart option

## 🛠 Technologies Used

- Python
- Tkinter
- Random module

## 🧠 Concepts Used

- While loop
- If-else
- Functions
- String handling
- GUI programming
